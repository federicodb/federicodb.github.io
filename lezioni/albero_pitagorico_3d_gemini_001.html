Costruzione Geometrica

L'albero si basa sul Teorema di Pitagora. Se disegniamo un triangolo rettangolo sopra un quadrato di lato $L$, i due cateti saranno i lati dei quadrati successivi.

Lato_Destro = Lato * sin(Œ±)

Pseudocodice Ricorsivo

funzione ramo(lato, livello):
    // 1. Caso Base
    se livello == 0: stop

    // 2. Disegna Quadrato
    disegna(quadrato, lato)

    // 3. Calcola Nuovi Lati
    latoSx = lato * cos(angolo)
    latoDx = lato * sin(angolo)

    // 4. Ricorsione Sinistra
    sposta(angolo_alto_sx)
    ruota(angolo)
    ramo(latoSx, livello-1)

    // 5. Ricorsione Destra
    sposta(angolo_alto_dx)
    ruota(angolo - 90¬∞)
    ramo(latoDx, livello-1)

Nota Tecnica

In questa simulazione usiamo Three.js. Invece di disegnare pixel, creiamo oggetti 3D (`Mesh`) e usiamo i `Group` per gestire le trasformazioni nidificate (la rotazione del padre sposta automaticamente tutti i figli).
